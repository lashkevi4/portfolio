Техническое Задание: Создание Календаря на React
Основные компоненты:

Компонент Календарь (Calendar): Это корневой компонент приложения, который содержит состояние текущего дня, месяца и года. Он также отвечает за управление функционалом переключения месяцев и рендерит другие компоненты (Навигация, Неделя и Даты).

Компонент Навигация (Navigation): Этот компонент отображает текущий месяц и год, а также кнопки для навигации вперед и назад по месяцам. Он должен принимать в качестве пропсов текущий месяц, год и функции для изменения месяца.

Компонент Неделя (Week): Отображает дни недели. Это статический компонент, который не требует пропсов.

Компонент Даты (Dates): Этот компонент отображает даты текущего месяца в формате календаря. Он должен принимать в качестве пропсов текущий месяц, год и текущую дату.

Требования к функционалу:

При загрузке приложения текущий месяц, год и день должны быть установлены на сегодняшнюю дату. Эти значения должны храниться в состоянии компонента Календарь.
Компонент Навигация должен отображать текущий месяц и год, полученный из пропсов.
При клике на кнопки навигации в компоненте Навигация, месяц в состоянии компонента Календарь должен увеличиваться или уменьшаться. Это также должно обновлять отображаемый месяц и даты.
Компонент Неделя просто отображает дни недели без какой-либо дополнительной логики.
Компонент Даты должен корректно отображать даты текущего месяца в формате календаря, исходя из полученного месяца и года. Текущая дата должна быть выделена.
Компоненты должны быть стилизованы в соответствии с предоставленным CSS.
Технологии:

Использовать последнюю стабильную версию React.
Использовать CSS для стилизации.
Код должен быть написан с соблюдением стандартов ES6 и выше.
Весь код должен быть оформлен и комментирован в соответствии с лучшими практиками.
Для более детальной информации и уточнений, программисту следует обратиться к заказчику проекта.

BODY

Представляет основное содержание веб-страницы.

HEADER

Предназначен для создания шапки сайта или раздела.

NAV

Предназначен для блока навигации.

MAIN

Предназначен для основного содержимого документа.

SECTION

Предназначен для разделов контента.

ARTICLE

Предназначен для содержимого, которое можно независимо распределить или повторно использовать.

ASIDE

Предназначен для содержимого, слабо связанного с остальным содержимым страницы и которое может быть отдельно представлен.

FOOTER

Предназначен для создания подвала сайта или раздела.

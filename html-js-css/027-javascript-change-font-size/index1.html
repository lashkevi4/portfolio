Change font size with JavaScript

Note! All changes occur with JavaScipt. The font size of the list is changed by pressing the button below. The default font size for the list is 16px.

The font size of this list changes by pressing a button

  1. The first push of a button changes font size to 20px
  2. The second push of a button changes font size to 24px
  3. The third push of a button changes font size to 28px
  4. The fourth push of the button changes font size to 32px
  5. The fifth push of a button returns font size 16px
Current size: 16px

Change font type with JavaScript

Note! All changes occur with JavaScipt. The font type of the list is changed by pressing the button below. The default font on the list is "Courier New", font size 16px. But what happens if you press the button 10 times? The font type for this list changes from the button pressing

The font size of this list changes by pressing a button

  1. First push of a button changes font type to "Roboto"
  2. The second push of a button changes the font type to "Mansalva"
  3. The third Press of a button changes the font type to "Grenze"
  4. The Fourth press of a button changes the font type to "Turret Road"
  5. The Fifth push of the button returns the font type "Courier New"
